# Giới thiệu

Wiki cho **Valheim Dedicated Server** với mod **ServerCharacters** — lưu nhân vật trên server, chống mang đồ từ world khác.

## Hai cách deploy

| Cách | Thư mục | Phù hợp |
| --- | --- | --- |
| **Docker** | `./linux/setup.sh` | Tiện, tự update/backup |
| **Native** | `./native/setup.sh` | VPS RAM thấp (3 GB), có IP public |

## Bắt đầu nhanh

```bash
cd Valheim
cp .env.example .env
# Sửa .env: SERVER_PASS, SERVER_NAME, WORLD_NAME
./native/setup.sh
sudo ufw allow 2456:2457/udp
sudo systemctl start valheim
```

Thêm nội dung mới bằng cách tạo file `.md` hoặc `.mdx` trong thư mục `docs/`.
